Autonomous Solar‑Panel Cleaning Drone (PoC)

A FreeRTOS real‑time prototype running on ESP32 (Wokwi simulation). Demonstrates hard deadlines, ISR safety, mutex/queue sync, and deterministic motor control.

Project Summary

This prototype simulates an autonomous drone that cleans solar panels with a rotating brush and water spray. It demonstrates real‑time scheduling on an ESP32 using FreeRTOS: Motor Control (20ms), Sensor Fusion (50ms), and Obstacle Avoidance (30ms) are treated as hard real‑time tasks. Telemetry and a heartbeat LED run as soft real‑time tasks.

How to Run the Wokwi Demo

  1. Open the repository and click the Wokwi files (diagram.json and main.cpp) or import them into Wokwi.
  2. Start the simulator, open the Serial Monitor, and adjust the potentiometers to simulate dust, LiDAR distance, and battery.
  3. Press the Emergency Stop button to exercise the ISR and observe safe motor shutdown.

Included Files

  • main.cpp — FreeRTOS + ESP32 firmware used in Wokwi (motor control, sensors, telemetry)
  • diagram.json — Wokwi hardware diagram, wiring for potentiometers, LED, and buzzers
  • README.md — Project documentation and engineering analysis
